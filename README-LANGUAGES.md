# Language Files

This landing page is now available in two separate files:

## Files

- **index.html** - Vietnamese version (Phiên bản tiếng Việt)
  - URL: Default homepage
  - Language code: `vi`
  - Title: "Python cho Phân Tích Dữ Liệu - Khóa Học Data Science"

- **index-en.html** - English version
  - URL: /index-en.html
  - Language code: `en`
  - Title: "Python for Data Analysis - Master Data Science with Python"

## Language Switcher

Both files include language switcher links:
- Vietnamese version has an "EN" button linking to index-en.html
- English version has a "VN" button linking to index.html

The buttons are available in:
- Desktop navigation (top right)
- Mobile navigation (next to theme toggle)
